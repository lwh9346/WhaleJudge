# Whale Judger

Whale Judger是一个由go编写，基于docker的编程测评平台。

Whale Judger目前支持将golang作为答案的语言来做题，未来将支持cpp、python等语言。

开发者的英文水平极低，以至于名字写错了，不应该是"Judger"，而应该是"Judge"，这个问题会在后端开发完毕后修正

## 部署（目前仍不能部署）

1. 安装docker
2. 安装golang(>=1.15.2)
3. 克隆本仓库
4. 执行`go build`
5. 执行`WhaleJudger init`以初始化
6. 编辑配置文件
7. 执行`WhaleJudger`以运行http服务器
